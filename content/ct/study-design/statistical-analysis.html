DOSAGE REGIMEN

To control for type 1 error, primary and key secondary endpoints were tested sequentially using a step down approach.

Statistical significance (P <0.05) could be claimed for the next endpoint only if the prior endpoint in the sequence met significance requirements.

The time to event analyses (e.g., time to flare) used Kaplan-Meier methodology and the log rank test.
